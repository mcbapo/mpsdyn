nnpert reads a Hamiltonian <NNHamiltonian> from a text file, finds its ground state and uses as initial state for evolution after applying a local perturbation on the central site. If the total length is even, the perturbation is applied on the site to the right of the center of the chain.

Receives arguments:

Parameters
<infname>(char*) name of the input file with NNHamiltonian
<infnameP>(char*) name of the input file with perturbation
<D>(int) maximum bond dimension to be used
<stepBl>(int) periodicity for saving results of time evolution
<timefname>(char*) name of the input file containing the time configuration (tot nr of steps, list of delta and nrs)
<outfname>(char*) name of the output file
<app>(int) whether the output file is to be kept (app==1)
