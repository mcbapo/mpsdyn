Detailed Description

Implementation of a PXP Hamiltonian,

The class also implements a projector onto the constrained subspace (annihilating states with two excitations next to each other). If a penalty lambda is provided, the Hamiltonian includes a penalty term associating that value to any pair of neighbouring excitations. This can be used instead of the projector for some purposes.
