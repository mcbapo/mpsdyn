Detailed Description

Driver program for imaginary time evoltuion of the truncated SU(2) Hamiltonian with staggered fermions from Erez, Ignacio and Benni

Parameters
<N>(int) Number of Fermions
<D>(int) Bond dimension
<epsilon>(double) Hopping parameter
<mu>(double) Mass
<g>(double) Coupling constant
<dt>(double) Time step size
<name>(string) Name of a potential ground state file (optional)
<leng>(int) length of the string, if 0 is given no string is placed
Author
Stefan Kühn
Date
16/03/2015
