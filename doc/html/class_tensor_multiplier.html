Detailed Description

an MPO, which, instead of being applied using TN techniques, is multiplied exactly tensor by tensor.

Multiplier interface with an optimized operation: product of three connected tensors

returning the product, substracts the projection of the argument onto a given vector, with specified weight. This is used by Contractor to find the best MPS approximation to an excited state, by projecting out the ground state.
