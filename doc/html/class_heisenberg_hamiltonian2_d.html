Detailed Description

Implementation of Heisenberg Hamiltonian with local magnetic field and translationally invariant coefficients, on a 2D system with a ladder geometry.

Notice it is defined in terms of spin matrices, not sigmas.

Because in this geometry, the Hamiltonian is no longer nearest-neighbour, there are more than two terms in the Trotter deccomposition of the evolution operator. There will be two (even-odd) horizontal terms and one (if there are only two legs) or two vertical ones. The following functions construct them. They take as argument the complex factor in the exponential, delta. For real time evolution, delta is -i*tau, for imaginary time, it should be -tau.
