Get the MPO for the exponential of the Hamiltonian times a certain coefficient, which can be real (-delta, negative, for imaginary time evolution) if imag=true, or imaginary (-i delta), if imag=false. It returns the MPO corresponding to second order Trotter with the decomposition H2, H1. If orderT!=2, not supported.
