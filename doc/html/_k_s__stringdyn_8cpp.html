Detailed Description

Program to look at certain string dynamics of the truncated Kogut Susskind Hamiltonian

Parameters
<N>(int) Number of Fermions
<d>(int) Physical dimension of Bosons in between
<D>(int) Bond dimension
<µ>(double) Dimensionless constant of the Hamiltonian
<x>(double) Coupling constant
<lambda>(double) Constant for the penalty-term in the Hamiltonian
