Detailed Description

Implementation of Fermi-Hubbard Hamiltonian on a finite chain with L 
sites containing fermions with spin (after Jordan Wigner transformation, 
those sites contain two spin 1/2 systems each).
\f[
H=-t\sum_{j,\sigma=\uparrow \downarrow}\left (c_{j,\sigma}^{\dagger}c_{j+1,\sigma}

+\mathrm{h.c.} \right ) +U\sum_j n_{j\uparrow }n_{j\downarrow } \f]
