Detailed Description

Full gauge invariant version of the Schwinger Model yet with truncated bosonic dimension (the link fileds are APPROXIMATED up to the desired order of the exponential)

Parameters
<N>(int) Number of Fermions
<d>(int) Physical dimension of Bosons in between
<D>(int) Bond dimension
<µ>(double) Dimensionless constant of the Hamiltonian
<x>(double) Coupling constant
<e>(double) Constant of the Operator
<lambda>(double) Constant for the penalty-term in the Hamiltonian
<ig>(void) Variable to determine if inital guess for the ground state should be read from HDD (optional)
<ac>(double) Accuracy for the contractor up to which the ground state should be calculated (optional)
<O>(unsigned int) Order up to which the exponential of the link fields is approximated by its series representation, the default value is 1 (optional)
