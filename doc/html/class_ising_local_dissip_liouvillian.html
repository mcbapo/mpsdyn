Detailed Description

Program isingDissipative: Find the steady state of an Ising chain with local dissipation, as specified in the model

The program requires a first argument with the name of a Properties file. For instance, projects/banulsm/config/dissipIsing.conf (vu also an empty file would work)

To replace some of the values (or provide missing ones) in the file by a command line option, the additional arguments have to be of the form -propName=propValue Accepted properties are:

  • L (int) length of the chain
  • D (int) maximum bond dimension
  • V (double) parameter of the Hamiltonian
  • Omega (double) parameter of the Hamiltonian
  • Delta (double) parameter of the Hamiltonian
  • gamma (double) parameter of the dissipation
  • scale (double) scale factor to multiply Lmpo
  • tol (double) tolerance asked from Contractor
  • mpsfile (string) file where the resulting MPS will be stored.
  • mpsdir (string) directory where to store such MPS files (default .)
  • oldmpsfile [OPTIONAL] (string) name of another (binary) file, containing an earlier MPS to be used as starting point. If it is "auto", the program searches for automatically created files in the mpsdir
  • outdir (string) directory where to write the (automatically named) file(s) containing observables.
  • wurounds [OPTIONAL] (int) Number of warm up rounds for the D=1 optimization
  • mode [OPTIONAL] (int) initial product state to be used: 0 (default) - prod |00> 1 - prod |10> (Z- for d=2) 2 - prod |01+10> 3 - prod |00+11> 4 - random reflection symmetric 5 - random
  • saveTmp [OPTIONAL] (int) Frequency (in nr. rounds) to save intermediate results during the optimization.
  • allDs [OPTIONAL] (vector of int) List of all values of D we want to try (check example properties)
