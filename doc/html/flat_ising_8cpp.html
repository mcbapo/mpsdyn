Detailed Description

Evolve Ising ground state after a local perturbation, to compare to results from nnpert.cpp Compute also entropy and spectrum of the rdm at the center.

flatIsing runs the findGroundState routine with the MPO for the Ising Hamiltonian <IsingHamiltonian>, then applies a local perturbation and evolves in real time for M steps using width delta. In the output file, we write the magnetization site by site as a function of time. In the second file, entropies are written across every cut.

Receives arguments:

Parameters
<L>(int) length of the chain
<J>(double) parameter of the Hamiltonian
<g>(double) parameter of the Hamiltonian
<h>(double) parameter of the Hamiltonian
<D>(int) maximum bond dimension
<M>(int) number of time steps
<stepBlock>(int) frequency (number of time steps) of writing
<delta>(double) step width
<outfname>(char*) name of the output file for the energy
<app>(int) whether the output file is to be kept (app==1)
<outfname2>(char*) name of the output file for the entropy
<app2>(int) whether the output file is to be kept (app==1)
<outfname3>(char*) name of the output file for the eigenvalues
<app3>(int) whether the output file is to be kept (app3==1)
