Detailed Description

This class is designed to be initialized at the beginning of the program from one or more files, and to store the values of the different arguments/parameters passed to the program as a configuration file. Then the main program will recover the needed parameters (using the standard names also included in the config file) from the Properties table, instead of requiring dozens of inputs in the command line.
