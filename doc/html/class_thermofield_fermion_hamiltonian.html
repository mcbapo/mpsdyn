Detailed Description

A Hamiltonian containing only nearest-neighbor terms, for the particular case of the thermofield simulations with fermionic bath. This means we have two fermionic modes in the center of the chain, and L fermionic modes coupled to it on each side. For the MPS implementation, a Jordan Wigner transformation is done to write the Hamiltonian in spin language.
